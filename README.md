# LUMINA Fashion Studio

LUMINA is an AI-powered fashion experience that transforms user selfies into high-fashion editorial shots and generates dynamic "GlamCam" video content.

## Setup

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Set up Environment Variables:**
    Create a `.env` file in this directory:
    ```env
    VITE_GEMINI_API_KEY=your_api_key_here
    ```

3.  **Run:**
    ```bash
    npm run dev
    ```
